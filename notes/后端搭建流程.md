
# 创建Nest项目
```markdown
nest new backend -s -g -p pnpm
```
# 安装所需依赖
```markdown
pnpm i @nestjs/config @nestjs/jwt @nestjs/passport passport passport-jwt bcrypt prisma @prisma/client redis ioredis
```

- `@nestjs/jwt` + `passport-jwt`用于JWT登录认证
- `bcrypt` 密码加密
- `prisma` + `@prisma/client` 数据库ORM
- `redis` + `ioredis` 缓存
